# Proximity Alarm
The alarm will activate the Green led :green_circle: if we are 30 centimeters away, otherwise the Red led will blink :red_circle:.

## Components
- 1 led Green
- 1 led Red
- 2 Resistors (220Î©)
- Ultrasonic sensor HC-SR04
